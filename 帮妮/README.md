🎀 帮妮 · HelpGirls
初心相助，即时响应 —— 打造女性身边的「数字安全庇护所」

一款基于位置服务的即时女性互助工具。结合 Web3 链上存证与去中心化存储，解决突发状况下的求助难题，建立一个真实、安全、可信的女性互助网络。

🌊 项目信息
参赛赛道：赛道3

核心价值：即时互助 · 数据不可篡改 · 女性信任安全网

产品定位：高响应、低门槛的女性紧急求助与社交平台

技术栈：HTML/CSS/JS + LBS 定位服务 + 智能合约（奖励与身份） + 去中心化存储（帖子存证）

部署网络：Avalanche 测试网（或其他兼容网络）

🎯 痛点与背景
突发困境无援：如「经血倒赔事件」等突发社交尴尬或困境，缺乏即时寻找附近同性的渠道。

恶意举报与封禁：女性发出的真实呼吁或互助帖常因平台审查被误删。

信任缺失与滥用：传统求助平台易被虚假信息充斥。

致敬「小艾帮帮」：延续即时呼应的精神，利用 Web3 技术解决信息留存与信任激励问题。

✨ 产品设计用心处 
温情安抚机制：模拟接通时不仅有位置确认，还加入心理安抚语音，缓解求助者的焦虑。

严格反滥用逻辑：每日限制 1 次免费呼助，第二次需申请，确保资源的紧急性与稀缺性。

极速上手引导：30 秒新手教学，确保在紧急时刻用户能瞬间理解如何操作。

安全闭环：删除“我很害怕”等负面暗示选项，通话结束后强制语音确认安全状态。

🏗️ 核心功能
1. 即时呼救 (SOS Response)
地理围栏：一键向附近 1-3 公里的「帮妮」用户发起求助。

即时响应：显示附近志愿者位置，确认接单后建立实时音频/文字连接。

2. 帖子内容上链 (On-chain Social)
数据主权：互助帖子与记录存储在链上（或通过存储地址关联），即便中心化前端删除，用户仍可恢复记录。

女性友好治理：去中心化管理员机制，专门清理对女性不友好的言论，保护社区氛围。

3. 信任与奖励机制 (Incentive)
帮妮勋章：每一次成功的帮助记录都会转化为链上凭证，作为信用背书。

爱心回馈：在保护隐私的前提下，奖励积极助人者（积分或 SBT），形成正向循环。

4. 庇护所功能（待定/愿景）
安全地图：标记附近的女性友好空间或商家作为临时避难点。

🔧 技术架构
定位系统：基于浏览器/手机 LBS 获取实时位置。

存储层：使用去中心化协议（如 IPFS/Filecoin）备份帖子内容，确保信息不因审查消失。

合约层：管理用户求助次数、存储互助证明及发放奖励。

前端：简洁、高对比度的紧急操作界面，淡粉或柔和色调。

🧩 创新亮点
真正的 SocialFi 落地：将社交与实际生存需求结合，不仅是聊天，而是救急。

反审查特性：利用 Web3 解决女性叙事被抹除的痛点。

极简 Web3 入门：用户甚至感知不到钱包，只需通过社交账号或简单操作即可完成上链。

📌 核心价值
