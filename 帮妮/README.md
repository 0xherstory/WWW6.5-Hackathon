1️⃣ 即时呼救 (SOS Response)
地理围栏

一键向附近 1-3 公里的「帮妮」用户发起求助还有自行设定的紧急联系人，还有一键报警功能
智能地理围栏自动筛选可帮助的用户
精准、快速的地理定位匹配

即时响应

显示附近志愿者位置
确认接单后建立实时音频/文字连接
支持匿名化的身份保护

用户点击SOS → 地理定位 → 发送求助信号 → 
附近用户收到通知 → 确认接单 → 实时连接 → 
完成互助 → 链上记录 → 评价反馈

信息永久存在，不可篡改

女性友好治理
