# SheProves · 她证

> 用她的方式理解数学，用她的语言教会彼此，用她的能力赚到第一笔钱。

---

## 这是什么

SheProves 是一个面向高中女生的数学互助与知识变现平台。

核心机制：**一个女生用自己的方式讲明白一个数学知识点，另一个女生投票"我听懂了"。讲得好的人赚到钱，听懂的人花了远低于家教的成本。**

不是刷题，不是网课，不是标准解法。而是：

- 用烘焙配方比例讲等比数列
- 用追星时间线画函数图像
- 用分手概率讲条件概率

每一条被社区验证的讲解，永久记录在 Avalanche 链上，归属于创作者。

---

## 解决什么问题

**对学的人：** 数学补课 ¥200/小时，但不一定听得懂老师的讲法。在 SheProves 上花 ¥5 发个悬赏，得到 3-5 个同龄人用不同角度讲同一个知识点——更便宜，可能更容易听懂。

**对讲的人：** 你明明数学很好，但高中生没有任何途径用这个能力赚钱。在 SheProves 上，你用自己的话讲明白一道题，就能拿到悬赏奖金和打赏。能力强的女生月入几百到上千。

**对教培品牌：** 获客成本越来越高。在 SheProves 上赞助悬赏，同时获得品牌曝光、优质 UGC 内容、和最会讲课的高中女生的招募通道。

## 经济模型

### 钱怎么流

```
求助者 ──付 AVAX──→ 悬赏合约 ──90%──→ 最佳讲解者
                         └──10%──→ 平台

读者 ──打赏 AVAX──→ 打赏合约 ──95%──→ 讲解者
                         └──5%──→ 平台

教培品牌 ──付 AVAX──→ 品牌悬赏池 ──奖金──→ 获奖者
           └──合作费──→ 平台
```

### 三个赚钱方式

#### 1. 悬赏讲解（主力）

求助者发帖："谁能用人话讲明白参数方程？"，附 AVAX 悬赏锁入合约。多位女生提交讲解，社区投票"听懂了"。求助者选择最佳讲解，合约自动分账：90% 给讲解者，10% 给平台。48 小时超时则得票最高者自动获奖。

#### 2. 打赏

看到好讲解随手打赏。95% 归讲解者，5% 归平台。

#### 3. 品牌悬赏

教培机构出资发起主题悬赏（如"用最有创意的方式讲圆锥曲线"），获奖者拿奖金 + 品牌兼职讲师机会。品牌获得曝光和人才发现，平台收取合作费。

### AVAX vs $PROVE：钱和声誉分开

| | AVAX | $PROVE |
|--|------|--------|
| 性质 | 真金白银 | 教学声誉（不可交易） |
| 获得方式 | 悬赏奖金、打赏 | 讲解被验证、投票、审核 |
| 用途 | 发悬赏、打赏 | 解锁角色权限、增加可信度 |

$PROVE 高的女生悬赏回答排名靠前、被品牌优先签约、声誉链上可查用于升学申请。

## 为什么上链

**1. 悬赏资金安全。** AVAX 锁在合约里，不经过平台账户。选中最佳回答后自动分账，杜绝平台挪用。

**2. 教学能力不可伪造。** "312 人投票说听懂了"这个数据在链上，任何人可验证。不是平台数据库里的数字，是 Avalanche 网络共识确认的事实。

**3. 收入记录透明。** 每笔悬赏、打赏的金额和流向链上公开可查。女生可以向家长证明"我靠讲数学赚了钱"。

**4. "女生能教数学"变成公共数据。** STEMIndex 合约实时记录平台上有多少女生在贡献讲解、多少被验证有效。这个数据任何人可从链上读取。

## 为什么是 Avalanche

### C-Chain 上的实际优势

每一次"听懂了"投票、每一笔打赏都是独立的链上交易。活跃社区意味着每天数千次写入。

- Gas < $0.01/tx → ¥0.01 的打赏也经济可行
- 亚秒确认 → 投票和打赏即时到账，体验流畅

### Subnet 愿景：教学即挖矿

Avalanche 的 Subnet 允许创建自定义链，定义谁是验证者、用什么付 gas。

SheProves Education Subnet 的设计：

| 维度 | 普通链 | SheProves Subnet |
|------|--------|-----------------|
| 验证者资格 | 质押资金 | 教学声誉 Top 100（$PROVE 最高的用户） |
| Gas 代币 | AVAX | $PROVE（通过教学赚取，不用购买） |
| 共识含义 | 交易有效性 | 讲解的科学准确性 |

在这条链上，**教数学的能力就是算力**。当前 MVP 在 C-Chain 上用合约模拟这套机制，未来迁移到独立 Subnet。

## 技术架构

### 合约（Avalanche Fuji Testnet）

| 合约 | 功能 |
|------|------|
| `ExplanationManager.sol` | 提交讲解、投票"听懂了"、悬赏系统（创建/回答/选择/超时自动结算）、打赏、角色升级（Learner → Reviewer → Validator） |
| `ProveToken.sol` | ERC-20 声誉代币，不可交易，由 ExplanationManager 铸造 |
| `STEMIndex.sol` | 链上聚合统计：参与人数、讲解数、验证率、悬赏总额、打赏总额 |
| `SheProvesCredential.sol` | Soulbound ERC-721 成就凭证 |

### SBT 成就

| 成就 | 触发条件 |
|------|---------|
| First Voice | 首条讲解被验证 |
| First Earning | 首次赚到 AVAX |
| Math Translator | 10 条讲解被验证 |
| Top Earner | 累计赚取 ≥ 1 AVAX |
| Trusted Reviewer | 验证他人 50 次 |
| Viral Explainer | 单条讲解 100+ "听懂了" |
| Brand Pick | 被品牌悬赏选中 |
